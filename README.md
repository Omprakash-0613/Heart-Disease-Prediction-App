# ❤️ Heart Disease Prediction Web App

## Overview
This is a Machine Learning web application built using:

- RandomForest Classifier
- Streamlit
- UCI Cleveland Heart Disease Dataset

The model predicts whether a patient is at risk of heart disease based on medical parameters.

---

## Model Performance
- Test Accuracy: 83.6%
- Algorithm: Random Forest
- Dataset Size: 303 samples

---

## Disclaimer
⚠️ This model is trained on a small academic dataset and is for educational purposes only.
It is NOT a medical diagnosis tool.

---

## Features
- User-friendly interface
- Risk probability output
- Risk level classification (Low / Moderate / High)
- Ethical medical disclaimer

---

## Deployment
This project is deployed using Render.
